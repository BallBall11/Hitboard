### Hitboard Zero PCB（电路板） 说明
#### 1. 使用整块 PCB 作为电路

根据你GH60外壳来决定参考哪个PCB设计，如果是默认的左上角 typec 开口，就用「typec 左上开口」。
如果自己挖洞走居中的 typec 开口，就用「typec 居中开口」。

// 由于整块 PCB 可以固定所有 pico、轴座，不再需要打印「03 轴座板」

#### 2. 使用亚克力和飞线作为电路

「03 轴座板」左上角设计了一个 48x48 的缺角。只需要使用嘉立创免费券，打印一个 48x48 的功能键电路板，就可以替代掉之前铜箔胶带贴电路的操作，更简单、更稳定。

// 推荐最开始使用飞线方案。因为把 Pico 焊接到 PCB 之后，几乎无法无伤拆解。等验证找到适合自己的布局之后，再考虑 PCB。

// PCB 相比飞线，所需要的焊点数量是一样的，并不会省时间。PCB 制造耗能、耗材料，不环保，不建议使用大块的 PCB。
