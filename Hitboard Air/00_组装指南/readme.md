### Hitboard Air - 组装指南
0. 根据「HitboardAir 物料清单」采购对应的元器件

1. 给 Pico 刷入 GP2040 固件

    按住 Pico 复位按钮的同时，通过 typec 数据线连接 Pico 到电脑。此时 Pico 将会以 U 盘的形式出现。
    将固件「GP2040-CE_0.7.12_Pico.uf2」文件，拖拽到 Pico U 盘。
    等待几秒钟，Pico 会被重新识别为手柄。可以在手柄测试网页 https://hardwaretester.com/gamepad 查看手柄状态，用导线短接 GND 和任意信号引脚查看是否工作。

2. 飞线连接元器件

    物理卡位：拿出「Hitboard Air 03 线路板」，板子上布满卡扣，依次将 Pico、轴座卡在对应的位置。
    熟悉电路图：打开「Pico 正面」的引脚示意图，了解引脚的基本情况。（组装时看的是「Pico 背面」，留意引脚编号）
    飞线连接：
        从 Pico 的任意 GND 焊盘（圆孔），引出导线，将导线依次通过每一个轴座的某个引脚（轴座有 2 个引脚任选一个），焊接确保联通（或者缠绕连接）。这样就完成了按键的接地。（功能键同理）
        将按键轴座另外一个引脚，接到对应的功能。比如方向键「Up」连接到 GP2。
        把 4 个方向键、8 个功能键、Select、Start 连到 Pico 对应的引脚。
        三个加键连到 Pico 的 18、19、20 引脚。//这三个引脚默认是 LS、RS、Home 功能，后续需要在设置中，改为你需要的其他按键。
        
        // 连接完成后，务必把 Pico 连接到电脑，用导线短接每个轴座的两个引脚。在手柄测试网页查看是否都能生效。
        // 如果不生效，检查是否虚焊，直到所有按键电路都正常工作。
        
3. 组装所有板子

    拿出「Hitboard Air 04 外壳」，将 03、02，依次放入外壳。用 5mm 螺丝固定。//左上、右下的螺丝留给盖板。
    将轴体压入 02 定位板。//可以用手指挤压轴体左侧的外壳，让轴体不损耗卡扣
    放置壁纸（如果有的话）
    放置功能键按钮
    放置盖板。拧盖板螺丝。
    
    
按键设置：

    按住 Start 按键的同时，通过 typec 连接电脑。
    访问 GP2040 Web 配置界面（通过 USB 连接后访问 192.168.7.1）
    点击「功能配置-GPIO 引脚配置」，将 LS、RS、Home 等按键，改为需要的按键。
        点击引脚查看器，按下需要配置的按键，网页会显示引脚编号（如果第一次检测无效就关闭弹窗再次点击）。
        在按键列表中找到对应的引脚，编辑按键即可。

手柄性能设置：//必做

    点击「设置-游戏手柄设置」。
    将 SOCD 清理模式，改为「后输入优先」。
    将去抖延迟，改为 0 毫秒。
    
    
恢复出厂设置：

    取下盖板，通过定位板的复位孔，按 Pico 复位按钮的同时，通过 typec 数据线连接 Pico 到电脑。此时 Pico 将会以 U 盘的形式出现。
    将固件「GP2040-CE_0.7.12_Pico.uf2」文件，拖拽到 Pico U 盘。完成刷机。
