### HITBOARD 开源项目

Hitboard 是一个专为键盘玩家设计的、高度紧凑、可完全自造的街机控制器（Hitbox）开源项目。它诞生于一个简单的需求：为何不能有一台既拥有键盘的紧凑布局，又具备 Hitbox 专业输入体验的设备？

本项目不仅提供可生产的文件，更完整记录了从 设计思路、CAD布局、PCB工业化 到 固件配置 的全流程。你可以直接使用，也可以将其作为起点，打造属于你自己的终极武器。

#### ✨ 项目特点

键盘布局，零学习成本：基于 GH60 键盘壳尺寸，沿用最熟悉的打字手位，上手即用。

全开源可定制：提供所有原始设计文件（CAD/PCB），你可以自由调整每一个按键的位置。

工业化 PCB 设计：采用专业 PCB 替代飞线，集成 WS2812B LED 灯带支持，稳定且易于焊接。

GP2040-CE 原生支持：完美兼容开源固件 GP2040-CE，享受持续的功能更新与强大的社区支持。

模块化设计：预留背光、热插拔摇杆模块接口，功能可自由增减。


#### ✨ 型号说明
Hitboard Zero
从零开始的起点，为渴望亲手定义操控的创造者而生。纯亚克力多层结构，只需简单修改CAD文件即可调整布局——方向键位置、按键间距、功能键分布，一切由你决定。这里仅提供非对称与对称两套参考布局作为灵感，而非定死的答案。288×96mm键盘尺寸无缝融入桌面，双面壁纸让美学随你变换。它是你打造专属控制器的空白画布，也是通往深度定制的第一级台阶。

![Hitboard Zero](./images/Hitboard_Zero.png)

Hitboard Air
为动手党和移动场景而生。基于3D打印技术实现，256mm宽度适配家用打印机。保留Zero的全尺寸按键布局，并提供更灵动的“拨片”加键。所有电路部分均为卡扣结构，可以方便的拆卸转移。拼 Hitbox 像拼四驱车一样简单。

![Hitboard Air](./images/Hitboard_Air.png)
