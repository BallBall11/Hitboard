### HITBOARD 开源项目

Hitboard 是一个专为键盘玩家设计的、高度紧凑、可完全自造的街机控制器（Hitbox）开源项目。它诞生于一个简单的需求：为何不能有一台既拥有键盘的紧凑布局，又具备 Hitbox 专业输入体验的设备？

本项目不仅提供可生产的文件，更完整记录了从 设计思路、CAD布局、PCB工业化 到 固件配置 的全流程。你可以直接使用，也可以将其作为起点，打造属于你自己的终极武器。

#### ✨ 项目特点

键盘布局，零学习成本：基于 GH60 键盘壳尺寸，沿用最熟悉的打字手位，上手即用。

全开源可定制：提供所有原始设计文件（CAD/PCB），你可以自由调整每一个按键的位置。

工业化 PCB 设计：采用专业 PCB 替代飞线，集成 WS2812B LED 灯带支持，稳定且易于焊接。

GP2040-CE 原生支持：完美兼容开源固件 GP2040-CE，享受持续的功能更新与强大的社区支持。

模块化设计：预留背光、热插拔摇杆模块接口，功能可自由增减。


### 使用指南

1. 确定布局

A4 纸打印「Hitboard Lite 布局.png」文件，看自己的手适不适用默认布局。

    如果适用，跳转到下一步。
    如果不适用，根据项目视频中的 CAD 文件设计、PCB 设计两个教程，设计自己的布局。
    
2. 采购物料

    拿 CAD 文件，找亚克力工厂，跟对方指定板子规格：
    
        01板子 5mm 磨砂亚克力  //白色壁纸建议用磨砂亚克力，黑色壁纸建议用透明亚克力
        02板子 2mm 磨砂亚克力
        04板子 2mm 磨砂亚克力
        
    拿 PCB Gerber 文件，在微信小程序搜：捷配。
    
        捷配首页领取每个月的免费券
        点击首页底部 Tab：免费打样
        输入板子规格：28.8cm 宽，9.6cm 高，3 片。下单
        点击首页底部 Tab：购物车，上传 Gerber 文件，结算  //文件传到微信，点上传后选择对话框中的文件就行，奶奶级用户也能 5 分钟下单
    
    拿壁纸文件，在照片打印店，打印 A4 尺寸照片
    
        店铺是全自动流程，下单后上传照片，等收货
        
    采购其他器件
    
        键帽  //DOIO 目前质感和外观综合胜出
        轴体  //只推荐猎隼 mini，其他轴都或多或少有硬伤
        轴座  //矮轴 1350 规格，跟店铺确认是猎隼 mini 能用的版本
        pico  //虽然国产便宜 5 块钱，还是推荐原版，避免因为芯片失灵报废整块电路板
        M2x12 螺丝
        导电胶按钮
        
    // LED 灯珠可以先不买。
    
3. 组装

    焊接：因为 PCB 上面器件的位置显而易见，把轴座、pico 焊接上去就行了。
    
    装盒：
    
        左右两个螺丝孔，固定定位板->PCB->支撑板->键盘壳。
        斜方向两个螺丝孔固定盖板。
        //这样在更换轴座的时候，只需要拧 2 颗盖板螺丝。
        //如果壁纸孔剪的大，更换壁纸也只需要打开盖板。
        
    调整按键键位：底部三个键依次为：L3、Up、R3，可根据自己的习惯调整。
    
        按住顶部功能键第三个按钮（select）的同时，插入 usb。浏览器访问 192.168.7.1，打开 GP2040 设置界面改键。
        
总结：

    组装步骤，全程不需要对线路之类的进行确认，操作时间在 1 小时左右。
    
    // 后续，LED 灯珠比较小，每个灯 4 个焊点。可能需要更久的时间。
    // 建议：先不装 LED 灯，玩起来再说。因为只有你玩街霸足够久，足够深入，才能发现你设备的哪些部分是需要调整的，做针对性优化。第二版，才是你的完美版本。
